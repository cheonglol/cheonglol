# Web Template

A modern full-stack web template built with Astro, React, Fastify, and Prisma.

## Stack

- **Frontend**: Astro 4 + React 18 + TanStack Query 5 + SASS
- **Backend**: Fastify 5 + Prisma 7 + oRPC-style routes
- **Database**: PostgreSQL (Railway; dev can use SQLite)
- **Package Manager**: Bun (workspaces)

## Features

- Responsive design
- Docker support
- API with rate limiting
- Database ORM with Prisma

## Getting Started

```bash
# Install dependencies
bun install

# Development
bun run dev

# Build
bun run build
```
